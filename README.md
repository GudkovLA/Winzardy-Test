# Прогресс по заданию:
- [ ] Игровой персонаж

    - [X] Имеет здоровье

    - [X] Может перемещаться с помощью **WASD**
    
        - [X] Может проходить сквозь препятствия
        
    - [X] Из персонажа раз в секунду в случайном направлении вылетает снаряд, который движется по прямой
    
        - [X] При попадании в противника, наносит ему 25 единиц урона
        
    - [ ] При нуле здоровья:
    
        - [ ] Возможность перемещать персонажа блокируется
        
        - [ ] Стрельба снарядами прекращается

- [X] 3D-камера, вид сверху, следует за персонажем игрока

- [ ] Противники
    - [X] Имеют 100 единиц здоровья
    - [ ] Раз в 7 секунд за пределами видимости камеры появляются противники
    - [X] Двигаются в сторону игрока
        - [ ] Останавливаются, когда подходят близко к игроку  
    - [X] Наносят игроку 5 единиц урона в секунду, когда находятся рядом с ним
    - [X] При нуле здоровья - умирают
    
- [X] Монетки
    - [X] При смерти противника с шансом 50% выпадает «Монетка»
    - [X] Когда игрок находится рядом с монеткой, она исчезает, а игроку добавляется +1 к значению ресурса
    
- [ ] UI
    - [X] В левом верхнем углу экрана, выводится цифра текущего здоровья игрока
    - [X] В правом верхнем углу экрана выводится значение ресурса игрока (Кол-во собранных монеток)
    - [X] Над противниками должны выводиться цифры здоровья (С помощью **Unity UI**)
    - [ ] После смерти персонажа открывается экран проигрыша с кнопкой перезапуска
